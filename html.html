HTML (HyperText Markup Language)

1. Budowa znaczników / tagów:

<znacznik atrybut=”wartość”>treść</znacznik>

2. Najważniejsze znaczniki / tagi:

<head>...</head> - ustawienia główne dokumentu niewidoczne na stronie,

<body>...</body> - część widoczna na stronie,

<main>...</main> - główna część dokumentu,

<header>...</header> - nagłówek dokumentu / sekcji / artykułu,

<footer>...< /footer> - stopka dokumentu / sekcji,

<h1 - h6>...< /h1 -h6> - poziomy nagłówków,

<section>...</section> - sekcja dokumentu,

<article>...< /article> - samodzielny artykuł,

<p>...< /p> - akapit / paragraf,

<br/> - przerwa w tekście przeniesienie do nowej linii,

<b>...< /b> - pogrubienie bez znaczenia,

<strong>...< /strong> - pogrubienie podkreślające ważność,

<i>... </i> - kursywa bez znaczenia,

<em>...</em> - kursywa podkręślająca ukryte inne znaczenie,

<element id="nazwa"> - nadawanie indywidualnej nazwy elementom,

<element class="nazwa"> - przydzielanie elementu do klasy,

<a href="target">...< /a> - link zewnętrzny,

<a href="#nazwa">...< /a> - link wewnętrzny w dokumencie,

<nav>...< /nav> - nawigacja w dokumencie,

<img src="plik.jpg" alt="tekst opisujący"/> - wstawianie obrazów,

<script>...< /script> - wstawianie skryptów do dokumentu.

3. BEM - Block Element Modifier:

block__element--modifier

Block - nadrzędny element,

Element - część bloku (nie musi być bezpośrednim dzieckiem),

Modifier - modyfikator - jeśli element występuje w kilku wariacjach tworzymy dodatkową klasę.

Stronę należy podzielić na bloki ("block"), wewnątrz których ustala się elementy składowe, każdemu z tych elementów należy nadać klasę ("block__element").
Jeśli element ma modyfikator ("block__element--modifier"), w modyfikatorze należy zmienić tylko unikalne dla niego wartości, nie należy przepisywać cech z klasy bazowej.
