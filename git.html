GIT

1. Konfiguracja

Konfiguracja:

git config --global user.name "Jan Nowak" - nazwa użytkownika,

git config --global user.email jannowak@example.com - adres e-mail użytkownika,

git config --get user.name - zwraca nazwę użytkownika,

git config --get user.email - zwraca adres e-mail użytkownika.

2. Podstawowe komendy:

git init - tworzy niewidoczny katalog w katalogu z projektem, który chcemy dodać do repozytorium,

git status - zwraca aktualny stan repozytorium oraz spis plików wchodzących w jego skład,

git add . - dodaje do repozytorium wszystkie pliki z głownego katalogu projektu,

git commit -m "opis zmian" - tworzy kopię aktualnego stanu projektu wraz ze spisem zmian dokonanych od ostatniego commitu,

gitk - wywołuje graficzny interface ukazujący wszystkie zapisane wersje i zmiany,

git push - wysyłanie ostatnich zmian na serwer GITHub.

3. GITHub:

Adres strony: GITHub

Po założeniu repozytorium na stronie, możemy wybrać jedną z 3 opcji dodania do niego plików i dla każdej z nich mamy zestaw komend do skopiowania i wklejenia do terminala w celu wykonania żądanej operacji. W terminalu pojawią się pytania o dane do logowania, które trzeba uzupełnić, by repozytorium online zostało zsynchronizowane z tym na dysku.
