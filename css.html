CSS (Cascading Style Sheets)

1. Budowa reguły:

selektor {
właściwość: wartość;
}

2. Link do zewnętrznego pliku ze stylami (.css):

<link rel="stylesheet" type="text/css" href="style.css">

3. Specyficzność stylu:

!important -> <znacznik style="właściwość: wartość">...</znacznik> -> #name {} -> .name {} -> element {}

!important - wstawione w definicję stylu eliminuje wszystkie nadpisania,

<znacznik style="właściwość: wartość">...</znacznik> - styl inline, wstawione w definicję elementu na stronie,

<znacznik class="name">...</znacznik> -> selektor: .name - przydział elementu do klasy o danym stylu,

<znacznik id="name">...</znacznik> -> selektor: #name - nadanie elementowi indywidualnej nazwy.
