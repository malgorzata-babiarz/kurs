JavaScript

1. Standard:

ECMAScript2015+ (ES6+)

- ECMAScript jest to ustandaryzowany przez organizację ECMA skryptowy język programowania,

- JavaScript implementuje rzeczy ze specyfikacji ECMAScript,

- Rozwój ECMAScript mocno przyspieszył przez szerokie wykorzystywanie JavaScriptu.

2. Link do zewnętrznego pliku ze skryptami (.js):

<script src="target">...</script>

3. Polecenia

Kod JS uruchamia się raz - po załadowaniu strony.

alert('...'); - wyskakujące okienko o treści "...",

console.log(name); - wpis w konsoli przeglądarki o treści równej wartości zmiennej name.

4. Literały:

backtick / backquote (`):

console.log(`Nazywam się ${name} i mam ${age} lat`); - wstawianie zmiennych do tekstu.

5. Definiowanie zmiennych

const - zmienna mająca stałe połączenie z jedną wartością (const name='xyz'; const age=36;),

let - zmienna, której połączenie z wartością można przepinać (let isHappy = true; isHappy = 'tak';).


Proste typy zmiennych:

string - tekst, otoczony ' lub " (zalecane '),

number - liczba,

bigint - liczba większa niż 2^53,

boolean - wartość logiczna true lub false,

null - wartość celowo niezdefiniowana,

undefined - brak definicji,

*symbol - tworzy "unikalne opakowanie".


typeof nazwaZmiennej - polecenie sprawdzające typ zmiennej

6. DOM - Document Object Model

DOM to "złożony" w przeglądarce HTML.

JavaScript może modyfikować DOM:

document.querySelector('.selector') - zwraca node (węzeł) z dokumentu HTML; gdy selektor jest zapisany zgodnie z css, pozwala wyszukiwać elementy dokumentu po klasie (do pierwszego znalezionego pasującego elementu); możliwe przypisanie ich do zmiennych jako wartości.

nazwaZmiennej.innerHTML - pozwala na nowo zdefiniować lub zobaczyć wartość zmiennej przypisaną do niej w danym miejscu w skrypcie.

Korzystając z BEM warto tworzyć dodatkową klasę z modyfikatorem --js (ułatwia identyfikację / usuwa potencjalne problemy).
